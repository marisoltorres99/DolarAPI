# CurrencyConversionAPI

## Descripción
**CurrencyConversionAPI** es una API desarrollada en .NET que permite calcular la conversión de una cantidad ingresada en base a la cotización del tipo de dólar obtenida desde otra API. La API recibe un valor a convertir y lo multiplica por la cotización actual del dólar obtenida desde una fuente externa.

## Tecnologías utilizadas
- .NET 8
- ASP.NET Core
- C#
- Newtonsoft.Json
- HttpClient para consumo de APIs externas
